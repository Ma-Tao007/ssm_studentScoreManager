ssm的学生成绩管理系统

源码下载地址：http://www.manoncode.cn/details?id=52


框架：ssm+mysql+tomcat+easyui+jquery+ajax
1.学生：考勤管理，请假，选课，查询成绩，修改个人信息等

2.教师：考勤管理，请假审核等，选课信息管理，学生信息管理，班级信息管理，
课程信息管理，成绩管理，成绩导入导出功能，成绩统计（可视化展示），修改个人信息等

3.管理员：比教师多一个教师管理
